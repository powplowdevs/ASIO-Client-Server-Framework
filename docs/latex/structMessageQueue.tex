\doxysubsection{Detailed Description}
Manages a queue of messages and processes them asynchronously. 

The \mbox{\hyperlink{structMessageQueue}{Message\+Queue}} allows messages to be added to a queue and processed in a separate thread. It uses a condition variable to synchronize access to the queue.
