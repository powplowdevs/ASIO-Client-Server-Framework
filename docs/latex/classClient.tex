\doxysubsection{Detailed Description}
Represents a client that connects to a server, sends and receives messages asynchronously. 

This class manages the connection to the server, handles sending and receiving messages asynchronously using Boost.\+Asio, and provides a task queue to handle these operations in the background. \begin{Desc}
