\doxysubsection{Detailed Description}
Manages a queue of tasks and processes them asynchronously in a separate thread. 

The \mbox{\hyperlink{structTaskQueue}{Task\+Queue}} allows tasks to be added to a queue, which are then processed in a separate thread. It uses condition variables to manage synchronization between threads.
